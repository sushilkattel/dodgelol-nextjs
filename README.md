# DodgeLoL NextJS

Statistical guide and tracker for League Of Legends made using NextJS


## Used By

This project is used by the following repos:

- DodgeLoL server https://github.com/sushilkattel/dodgelol-server



## Deployment

To deploy this project run

```bash
  npm i -f
  npm run
```
